# Requirements
- Python 3.8.10
- Pip 20.0.2

## Configuration
- unpack archive with SMiLER data
```
cd data
tar -xzvf datasets.tar.gz
cd ..
```

- create virtual environment
```
python -m venv venv
source venv/bin/activate
```
